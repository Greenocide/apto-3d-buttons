Apto 3D Buttons

A reusable .apto-3d-button component with a flatter 3D edge, no drop shadows, icon states, toggles, and a loading to success flow.

Text buttons

Color presets

Preset color classes only set CSS variables, so they compose with size, icon, loading, and custom state classes.

Sizes

Icon + label

Icon-only

Icon buttons contain plain svg, i, and img elements. Put active color data on the icon itself, then data-apto-toggle stays on the button.

Toggle SVG stress test

These are all real toggle buttons using icon-level active paint values, including SVG-level and path-level data attributes.

Switch

A switch is just a button with apto-3d-button--switch and data-apto-toggle, so it stays accessible and shares the same active-state logic.

Loading to success

The same loading and success helper works for text buttons and icon-only submit buttons. The spinner and checkmark are pure CSS pseudo-elements, so no markup swaps are needed.
