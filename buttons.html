Apto 3D Buttons

Reusable 3D buttons, checkboxes, radios, switches, tabs, sliders, and scrollbars with a flatter edge, no drop shadows, native form behavior, icon states, and a loading to success flow.

Text buttons

Active text toggles

Text toggles use data-apto-toggle. Active buttons keep their normal background and text colors and remain physically pressed down.

Color presets

Preset color classes only set CSS variables, so they compose with size, icon, loading, and custom state classes.

Sizes

Icon + label

Icon-only

Icon buttons contain plain svg, i, and img elements. Put active color data on the icon itself, then data-apto-toggle stays on the button.

Toggle SVG stress test

These are all real toggle buttons using icon-level active paint values, including SVG-level and path-level data attributes.

Checkboxes

These use real checkbox inputs. The indeterminate example uses data-apto-tristate="mixed" and cycles through mixed, unchecked, and checked.

Switches

Switches are native checkboxes with role="switch". Their values submit with forms without any library-specific JavaScript.

Radio selection
